Amazon India Product Image Requirements

Main image basics: pure white background, sharp product, optimized file size.

Main image checklist (India sellers)

  • Background: RGB white (255,255,255) — no props or text on main image
  • Product fill: Product should occupy ~85% of frame
  • Format: JPG preferred; check Seller Central for latest specs
  • Quality: Sharp, no watermark on main image

Workflow with FastImageTool

  1. Resize so longest side meets Amazon minimum (typically 1000px+ for zoom — verify in Seller Central).
  2. Compress to reduce KB without visible blur.
  3. Use PNG to JPG if your studio export is PNG.

Secondary images

Lifestyle and infographic images can use different backgrounds — only the main image must be white.
