How to Use Test Tool

How to Use Test Tool

Follow these simple steps to use the Test Tool effectively:

  1. Upload your image by clicking the "Choose File" button.
  2. Select the desired settings or parameters for your test.
  3. Click the "Start Test" button to begin the analysis process.
  4. Wait a few moments while the tool processes your image.
  5. Review the results displayed on the screen.
  6. Download or save the results if needed.

That’s it! You’ve successfully used the Test Tool to analyze your image.
