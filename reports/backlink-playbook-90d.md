# 90-Day Backlink Playbook — FastImageTool

**Goal:** 20–50 referring domains in 90 days.  
**Tracker:** [backlink-tracker.csv](./backlink-tracker.csv)

## Week 1–2: Directories

Submit to: AlternativeTo, Product Hunt, SaaSHub, Toolify.ai, Futurepedia, Indie Hackers.

**Short pitch:** FastImageTool — 24 free browser-based image tools. No signup, files stay on device. Guides for Instagram, YouTube, Meesho, Amazon sellers.

## Week 3–6: Communities

| Channel | Angle |
|---------|-------|
| Reddit (seller/dev subs) | Meesho 600×600 guide, privacy-first tools |
| Meesho/Amazon Facebook groups | Link specific seller guides |
| Quora | Answer resize/compress questions with guide links |

Post structure: problem → solution → link to **one guide** (not homepage spam).

## Week 7–12: Outreach

Use `scripts/backlink_targets.csv` and `scripts/outreach_emails.csv`.

Linkable assets:
- https://fastimgtool.com/guides/resize-image-for-instagram.html
- https://fastimgtool.com/guides/compress-image-for-email.html
- https://fastimgtool.com/guides/meesho-product-image-size-600x600-under-50kb.html

**Target:** 2–4 new referring domains per week.

## Avoid

- Mass directory blasts in one day
- Paid PBN links
- Self-clicking AdSense
